# Distance-and-Similarity-Implement
an implement of several distances such as Euclidean distance, Chebyshev distance. 
## Usage
### Dependence
- python3
- numpy
- scipy
### Example
```
from distance import euclidean_distance as eucd
eucd(x, y) # x,y is n-dim vectors
```
